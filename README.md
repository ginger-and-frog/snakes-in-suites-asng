# snakes-in-suites-asng

The task here is to create a visually engaging, research-informed multimodal report that investigates a contemporary or historical instance of toxic organisational behaviour, unethical leadership and problematic followership of your choice. Please avoid picking ones we examine at length in class or those you have studied in other courses. This means no Elizabeth Holmes, Elon Musk, Adam Neumann, Travis Kalanick. Please don't work on the Boeing case-study, or the Volkswagen Diesel scandal either as these are covered in other courses.

The aim is to "unmask" the psychological, cultural, or structural dynamics behind the case, using concepts you have learned in the course. This means you MUST use the following sources: Hare and Babiak, Bakan, Harris, Kellerman and Padilla et al.

Here, the snake in suite is Osama Bin Laden.
